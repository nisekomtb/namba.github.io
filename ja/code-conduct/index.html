自然を尊重します：

  1. マウンテンバイク通行禁​​止の看板があるトレイル、又は許可されていないトレイルへは乗り入れないで下さい。
  2. 許可無くトレイルを作らないで下さい。
  3. 全てのライダーはコースアウトをしないで下さい。
  4. 常に正式なエントリーポイントからスタートして下さい。
  5. 雨が降った後等かの、ぬかるんだトレイルに立ち入らないで下さい。
  6. 自然保護の為に、トレイルに持ち込んだ物は全て持ち帰って下さい。
  7. 自転車を清潔に保ち、他の雑草や種を撒き散らかさないで下さい。
  8. 許可なく岩や丸太を移動したり、トレイルを変更したりしないで下さい。
  9. バイクをコントロールして、トレイル外に出ない様にして下さい。
  10. トレイル内では急なブレーキなどを避け、トレイルに損傷を与えない方法で乗車して下さい。
  11. メンテナンスのために閉鎖されたトレイルには立ち入らないで下さい。

他の人を尊重します：

  1. 他のトレイルユーザーを尊重して行動しましょう。
  2. 常に危険を予測して下さい。（誰かが次の角を曲がって来る可能性があること等）
  3. マウンテンバイク専用以外を使用する場合は、他のトレイルユーザーにあなたの存在を知らせて下さい。
  4. 追い越しをする時は原則し、充分注意して下さい。
  5. 常に注意して通過して下さい。
  6. なるべく、少人数で行動しましょう。
  7. 常に自分の速度をコントロールできる様にしましょう。

自分を尊重する：

  1. 常にヘルメットと保護具を着用して下さい。
  2. あなたの能力の範囲内で乗って下さい。
  3. 常に予見可能なリスクを予想して下さい。
  4. 事前にスケジュールを決めてから行動しましょう。

このガイドラインを守っていないトレイルユーザーを見かけた場合は、ガイドラインを守らないと、森林地帯や公有地でマウンテンバイクに乗れないことを丁寧に説明してください。

最後に、私たちのコミュニティの積極的なメンバーになること、トレイルのメンテナンス、イベント活動へのボランティア活動への参加を勧めて下さい。
