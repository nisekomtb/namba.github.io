ツインピークスバイクパーク：ガイド＆ツアーオペレーションについて

{% comment %}

ツイン ピークス バイク パークは一般の人々は無料で利用できますが、ツイン ピークス バイク パーク内で指導、ツアーなど、その他の金銭的授受を伴うサービスを提供する行為を行う場合、その個人または企業は、商業事業者パスを所有する必要があります。詳しくは商業事業者パスのページをご確認ください。

{% endcomment %}
